# 天九游戏 (Tianjiu Game)

一个基于 Go 和 HTML5 Canvas 实现的在线天九纸牌游戏。

## 游戏规则

### 牌型分类

#### 文子 (11种)
- 天、地、人、鹅、梅、长三、板凳、斧头、红头十、高脚七、铃铛六
- 每种2张，共22张

#### 黑子 (6种)
- 九(2张)、八(2张)、七(2张)、六公(1张)、五(2张)、生鸡(1张)
- 共10张

总计32张牌

### 牌型组合 (从大到小)

1. **至尊** (六公 + 生鸡) - 特殊牌，先出最大，否则最小
2. **文黑组合** (天九、地八、人七、鹅五) - 最大牌型
3. **双文子** (文子对子) - 如天天、地地等
4. **双黑子** (黑子对子) - 如九九、八八等
5. **3张牌组合** (文子2 + 黑子1 或 文子1 + 黑子2)
6. **4张牌组合** (文子2 + 黑子2)
7. **单张牌** (文子 > 黑子)

### 游戏玩法

- **玩家**: 4人游戏，1人庄家，3人平家
- **出牌顺序**: 顺时针，庄家先出
- **规则**:
  - 同一轮中，玩家必须出比前一个玩家更大的牌
  - 或者选择弃牌(睡牌)，丢弃相同数量的牌
  - 如果没有更大的牌，必须弃牌
  - 4个玩家都出完牌后，最大的牌赢得该轮
  - 赢牌的玩家在下一轮先出

## 安装和运行

### 前置要求
- Go 1.21+
- 现代浏览器 (支持 Canvas 和 WebSocket)

### 安装依赖

```bash
go mod download
```

### 运行服务器

```bash
go run main.go game_rules.go
```

服务器将在 `http://localhost:8080` 启动

### 访问游戏

在浏览器中打开 `http://localhost:8080`

## 使用说明

### 游戏界面

1. **中心区域** - 当前轮的出牌情况
2. **顶部** - 对家的手牌
3. **左右两侧** - 左家和右家的手牌
4. **底部** - 你的手牌，支持点击选择

### 操作

1. 点击你的手牌进行选择(选中的牌会高亮)
2. 点击"出牌"按钮提交选中的牌
3. 点击"弃牌"按钮放弃这一轮
4. 点击"清除选择"按钮重新选择
